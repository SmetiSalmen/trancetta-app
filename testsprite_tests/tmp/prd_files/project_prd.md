# Trancetta Pizza House PRD

## Product Overview
Trancetta Pizza House is a French-language pizza restaurant website focused on premium rectangular pizza sold by the slice or as a shareable tray.

## Primary Goals
- Present the restaurant brand and offering on the home page.
- Let users explore the menu and build a customized pizza tray.
- Provide location and contact information.

## Key User Flows
1. Visit the home page and scroll through the brand story, best sellers, gallery, and contact sections.
2. Navigate to the menu page.
3. Choose a tray size.
4. Drag pizzas into the available tray zones.
5. Review the order summary.

## Functional Requirements
- Home page must render all major sections: hero, concept, story, best sellers, why choose Trancetta, gallery, location, and contact.
- Menu page must support selecting a tray size.
- Menu page must support drag-and-drop pizza placement.
- Menu page must support a mobile pizza selector experience.
- Menu page must show an order summary for the current selection.

## Constraints
- The menu builder is client-side only.
- There is no checkout flow or backend order persistence.
- The site is optimized for a responsive, mobile-friendly experience.
